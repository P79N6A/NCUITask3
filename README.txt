Небольшая справка по программе :)
	1. Реализованы первый (дизайн, бэкенд, махинации с бд, сортировка по количеству страниц в книге) и второй(пагинация на сервере(по 5 книг)) пункты задания.
	2. При загрузке страницы приложения автоматически создается файл с небольшой базой данных в корневой папке Apache Tomcat
	3. При махинациях с бд (добавление, удаление, изменение элементов) каретка возвращается на первую страницу таблицы. Для удаления или изменения элемента,
	необходимо выделить его в бд :)
	4. Передвижение по таблице осуществляется с помощью кнопок:
		4.1 FirstPage - переход на первую страницу таблицы.
		4.2 PrevPage - переход на предыдущую страницу таблицы.
		4.3 NextPage - переход на следующую страницу таблицы.
		4.4 LastPage - переход на последнюю страницу таблицы
		4.5 ВНИМАНИЕ!!! ПЕРЕХОД С ПОСЛЕДНЕЙ СТРАНИЦЫ НА ПРЕДЫДУЩУЮ НЕ РАБОТАЕТ!!!(как и на следующую).!!!
		!!!В данном случае нужно вначале вернуть каретку на первую страницу.!!!
	5. В любой момент можно включить сортировку по количеству страниц у книг (CheckBox KolStr (включенный - отображение таблицы с сортировкой,
	нет - без сортировки)).
	5.1. ВНИМАНИЕ!!! Для того, чтобы изменения вступили в силу, необходимо сменить страницу таблицы.!!!
	6. Валидатор не пропускает строки с пробелами :)
